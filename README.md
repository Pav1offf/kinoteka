# KINOTEKA

Энциклопедия кино: Большая база данных о фильмах, сериалах и людях, связанных с ними (актёры, режиссёры, сценаристы и т.д.).
Не точная копия сайта кинопоиск, написана на React с использованнием API Kinopoisk.

![Главный экран приложения](public\screenshot-2.png "Демонстрация работы приложения")

## Возможности:

- Поиск кино по ключевым словам.
- Выбор фильмов по фильтрам(по странам, по годам, по жанрам), сортировка по рейтингу, по дате, по количеству оценок.
- Страница с деталями о фильме/сериале.
- Страница с деталями о актёрах, режиссёрах, сценаристах и т.д.
- Маршрутизация между различными страницами приложения при помощи React Router.
- Выбор темной или светлой темы.
